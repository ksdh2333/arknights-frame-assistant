## 下载及启用方法

1. 从 [GitHub Releases 页面](https://github.com/CloudTracey/arknights-frame-assistant/releases) 下载最新版本的 exe 文件
2. 与游戏一同启动（没有先后顺序要求）
3. 对所希望修改的按键和设置进行自定义
4. 点击**保存设置**或**应用设置**，即可在游戏中生效

## 功能特性

### 过帧与精细操作

| 功能 | 说明 |
|------|------|
| 一键技能 | 一键开启鼠标所指单位的技能 |
| 一键撤退 | 一键撤退鼠标所指单位 |
| 暂停选中 | 在暂停时选中鼠标所指的场上单位 |
| 暂停技能 | 在暂停时开启鼠标所指的场上单位的技能 |
| 暂停撤退 | 在暂停时撤退鼠标所指的场上单位 |
| 前进 33ms | 前进游戏 1 倍速下的 1 逻辑帧（游戏为 1 秒 30 逻辑帧） |
| 前进 166ms | 前进游戏 0.2 倍速下的 1 逻辑帧（选中单位或者待部署区时的 1 逻辑帧） |

> **注意**：由于目前 PC 端的输入限制，所有的暂停中操作均为有缝操作

### 快捷操作

| 功能 | 说明 |
|------|------|
| 模拟左键点击 | 模拟鼠标左键点击 |
| 放弃行动 | 放弃当前战斗行动（对应游戏内 V 键） |
| 跳过招募动画/剧情 | 点击跳过按钮，快速跳过招募动画或剧情 |
| 返回上级菜单 | 一键返回上级菜单（对应游戏内 ESC 键） |
| 基建快速收取 | 点击基建收取按钮，如快速收取制造站/贸易站产物 |
| 肉鸽收取道具 | 点击肉鸽模式的"收下"按钮，快速收取道具 |

### 卫戍协议操作

| 功能 | 说明 |
|------|------|
| 查看敌人 | 查看敌人详情（对应游戏内 W 键） |
| 调度中心 | 打开调度中心（对应游戏内 A 键） |
| 冻结 | 冻结单位（对应游戏内 S 键） |
| 刷新 | 刷新商店（对应游戏内 D 键） |
| 升级 | 升级商店（对应游戏内 G 键） |
| 出售/销毁 | 出售单位或销毁装备（对应游戏内 X 键和 I 键） |
| 准备就绪 | 准备就绪（对应游戏内 C 键） |
| 单位撤退 | 撤退单位（对应游戏内 Q 键） |
| 一键撤退 | 选中鼠标所指单位并撤退 |
| 一键出售/销毁 | 选中鼠标所指单位并出售/销毁 |
| 一键购买 | 快速双击购买鼠标所指单位 |

> **注意**：卫戍协议方案与常规作战、快捷操作方案互斥，切换到卫戍协议页面时其他按键将被禁用

### 全按键自定义

支持自定义绑定的功能包括：

**常规作战：**

| 功能 | 默认按键 | 功能 | 默认按键 |
|------|----------|------|----------|
| 按下暂停 | F | 前进 33ms | R |
| 松开暂停 | Space | 前进 166ms | T |
| 切换倍速 | D | 一键技能 | E |
| 暂停选中 | W | 一键撤退 | Q |
| 干员技能 | S | 暂停技能 | 鼠标前侧键 |
| 干员撤退 | A | 暂停撤退 | 鼠标后侧键 |

**快捷操作：**

| 功能 | 默认按键 | 功能 | 默认按键 |
|------|----------|------|----------|
| 模拟左键点击 | Z | 放弃行动 | 无 |
| 跳过招募动画/剧情 | 无 | 基建快速收取 | 无 |
| 返回上级菜单 | 无 | 肉鸽收取道具 | 无 |

**卫戍协议：**

| 功能 | 默认按键 | 功能 | 默认按键 |
|------|----------|------|----------|
| 查看敌人 | W | 升级 | G |
| 调度中心 | A | 出售/销毁 | X |
| 冻结 | S | 准备就绪 | C |
| 刷新 | D | 模拟左键点击 | 无 |
| 单位撤退 | Q | 一键撤退 | 无 |
| 一键出售/销毁 | 无 | 一键购买 | 无 |

- 支持绑定键盘上的绝大多数按键
- 支持绑定除鼠标左键外的鼠标按键（包括滚轮和侧键）
- 使用 `BACKSPACE` 可清除绑定

| 功能 | 说明 |
|------|------|
| **自动更新** | 支持自动检查并下载更新 |

## 注意事项

### 仅对明日方舟进程生效

本程序的按键功能仅对明日方舟进程（`Arknights.exe`）生效，不会影响游戏以外的操作。

### 游戏内按键需为默认设置

请确保游戏内的按键设置保持默认，否则可能导致按键功能不生效：

**常规作战：**
- 战斗内变速：D
- 释放技能：E
- 撤退干员：Q
- 放弃行动：V
- 暂停：空格

**卫戍协议：**
- 查看敌人：W
- 调度中心：A
- 冻结：S
- 刷新：D
- 准备就绪：C
- 升级：G
- 出售：X

### 关于游戏内帧率

- 如果你的屏幕刷新率**大于等于 120 Hz**：可以开启垂直同步，且小助手内的"游戏内帧率"可以设置为 120
- 如果你的屏幕刷新率**低于 120 Hz**：建议不要开启垂直同步，确保小助手内的"游戏内帧率"设置与游戏中实际帧数一致；若开启垂直同步，则需确保该设置与显示器刷新率一致

### 安全提示

本程序基于 AutoHotkey 开发。大部分 MMORPG 或竞技类网游的反作弊系统对 AutoHotkey 开发的程序非常敏感，可能将其误判为作弊程序，导致游戏无法启动甚至账号封禁。

**强烈建议开启"随游戏进程关闭自动退出"功能**（默认开启），避免因忘记关闭本工具而触发其他游戏的反作弊机制。

### 如果不小心关闭了设置界面

从电脑右下角托盘区（含隐藏图标）找到小助手图标，右键点击**打开设置**即可重新打开设置界面。

## 常见问题

### Q: 程序启动后没有反应？

A: 请检查以下几点：
1. 确认程序以管理员权限运行
2. 确认游戏已经启动
3. 检查右下角托盘区是否有小助手图标（包括隐藏的托盘区图标）
4. 检查是否在设置里关闭了"启动时打开设置窗口"

### Q: 按键绑定后不生效？

A: 请检查：
1. 游戏内按键设置是否为默认设置
2. 修改后是否应用或保存设置
3. 按键是否与其他软件冲突
4. 是否在游戏窗口内操作

### Q: 如何关闭自动更新检查？

A: 在设置界面的"其他设置"标签页中，取消勾选"自动检查更新"即可。

### Q: GitHub Token 是什么？需要配置吗？

A: GitHub Token 用于提高 API 请求配额。只有在提示 API 配额超限时才需要配置，一般情况下无需使用。

### Q: 如何彻底退出小助手？

A: 右键点击托盘区图标，选择"退出"即可。如果开启了**随游戏进程关闭自动退出**，游戏退出后 AFA 也会随之一同退出。
