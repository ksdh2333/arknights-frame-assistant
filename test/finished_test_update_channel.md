### 流程：测试版渠道更新检查

> 涉及模块：`updater/version_checker.ahk`、`updater/updater_manager.ahk`、`updater/updater_ui.ahk`

- [x] **前置**：`UpdateChannel = "2"`（测试版），网络正常
- [x] **操作**：点击"手动检查更新"
- [x] **预期**：调用 `/releases` API，遍历所有 release（含 pre-release）
- [x] **预期**：选出 SemVer 最高的版本，不区分正式版/测试版
- [x] **预期**：如果远程有比本地高的版本（含 pre-release），提示更新
- [x] **预期**：如果本地已是最新，提示"已是最新版本"

- [x] **异常**：测试版渠道下，GitHub API 返回空数组 → **预期**：提示"无法解析版本信息"
