## 集成测试

### 流程：正常更新下载流程

> 涉及模块：`downloader.ahk`、`updater_ui.ahk`、`updater_manager.ahk`、`version_checker.ahk`

- [x] **前置**：使用稳定网络环境，确保可访问 GitHub
- [x] **操作**：打开 AFA 设置窗口 → 切换到"其他设置"标签 → 点击"检查更新" → 若有更新，点击"下载更新"
- [x] **预期**：下载进度条正常推进，速度显示正常，下载完成后弹出安装提示
- [x] **预期**：下载过程中无崩溃、无卡死

### 流程：下载过程中取消

> 涉及模块：`downloader.ahk`、`updater_ui.ahk`、`eventbus.ahk`

- [x] **前置**：开始下载更新，下载进行中
- [x] **操作**：在下载进度条进行中，点击"取消"按钮
- [x] **预期**：下载立即中止，临时文件被清理，按钮恢复为"检查更新"状态
- [x] **预期**：无残留进程、无异常错误提示

### 流程：下载过程中关闭下载对话框

> 涉及模块：`downloader.ahk`、`updater_ui.ahk`

- [x] **前置**：开始下载更新，下载对话框显示中
- [x] **操作**：点击下载对话框右上角关闭按钮（或 Alt+F4）
- [x] **预期**：行为与点击"取消"一致——下载中止，状态正确重置
- [x] **预期**：关闭后重新打开设置窗口，无异常状态（按钮可正常点击）

### 流程：下载过程中点击手动下载

> 涉及模块：`downloader.ahk`、`updater_ui.ahk`

- [x] **前置**：开始下载更新，下载对话框显示中
- [x] **操作**：点击"手动下载"按钮
- [x] **预期**：浏览器打开手动下载页面（B站专栏），下载对话框**保持打开**，应用内下载继续进行
- [x] **预期**：应用内下载完成后正常弹出安装提示

##### 异常路径

- [x] **异常**：下载过程中断网 → **预期**：超时后显示错误提示，不崩溃，不残留无效 COM 对象引用
- [x] **异常**：下载过程中网络极慢（触发分块超时/停滞检测）→ **预期**：自动重试（最多 3 次），重试期间取消操作仍可正常中止下载
- [x] **异常**：连续快速点击取消按钮 → **预期**：不会重复调用 Abort 导致崩溃
