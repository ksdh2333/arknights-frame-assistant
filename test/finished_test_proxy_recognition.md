## 集成测试

### 流程：代理指挥开局 — 正常路径

> 涉及模块：`hotkey_actions`（`ActionBeginPause`、`TakeOverButtonPosition`）

- [x] **前置**：启用 AFA 开局自动暂停功能，进入游戏，设置代理指挥作战
- [x] **操作**：进入关卡，等待关卡开始
- [x] **预期**：游戏先暂停，随后立即取消暂停，游戏继续运行不处于暂停状态

### 流程：非代理指挥开局 — 正常路径

> 涉及模块：`hotkey_actions`（`ActionBeginPause`、`TakeOverButtonPosition`）

- [x] **前置**：启用 AFA 开局自动暂停功能，进入游戏，取消代理指挥作战，加载完成进入关卡
- [x] **操作**：进入关卡，等待关卡开始
- [x] **预期**：游戏暂停并保持暂停状态，不会自动取消暂停

##### 异常路径

- [x] **异常**：代理按钮透明度变化（开局动画期间） → **预期**：容错值 90 覆盖正常透明度波动
- [x] **异常**：不同分辨率、UI 缩放设置下代理按钮位置、大小不同 → **预期**：识别用图像与识别范围覆盖大部分位置、大小
- [x] **异常**：干员 Lancet-2（医疗小车）被误判为代理按钮 → **预期**：取消了点线识别，不再误判
