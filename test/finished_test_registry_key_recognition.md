# 测试清单：`注册表按键识别`

> 对应更改：新增 GameKeys 类（`game_keys.ahk`）从注册表动态读取游戏按键，替换 `hotkey_actions.ahk` 中所有硬编码游戏按键，支持用户自定义键位后 10 秒内自动适配。修复 ActionGameSpeed 不再误发送 G 键。`hotkey_control.ahk` 拦截正则改为动态生成。
