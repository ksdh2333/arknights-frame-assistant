## 集成测试

### 流程：默认隐藏到托盘

> 涉及模块：`GuiManager`、`Config`、设置保存/加载、托盘菜单

- [x] 前置：`ExitOnWindowClose=0`，设置窗口处于打开状态。
- [x] 操作：点击窗口右上角 X。
- [x] 预期：设置窗口隐藏，AFA 进程和托盘图标继续运行。
- [x] 操作：通过托盘菜单“打开设置界面”重新打开窗口。
- [x] 预期：窗口可以正常显示，设置内容不丢失。
- [x] 操作：按 Alt+F4。
- [x] 预期：行为与点击 X 相同。

### 流程：直接退出程序

> 涉及模块：`GuiManager`、`Config`、进程退出、托盘菜单

- [x] 前置：勾选选项并点击“应用设置”或“保存并关闭”，重新打开设置窗口。
- [x] 操作：点击窗口右上角 X。
- [x] 预期：AFA 进程和托盘图标退出，不残留设置窗口。
- [x] 重新启动 AFA 并打开设置窗口。
- [x] 操作：按 Alt+F4。
- [x] 预期：行为与点击 X 相同。

##### 异常路径

- [x] 勾选选项但不保存，点击 X。
- [x] 预期：按已保存的旧配置执行，不因未保存控件值改变关闭策略。
- [x] 已勾选选项时修改其他设置但不保存，点击 X。
- [x] 预期：程序直接退出，未保存设置不写入配置文件。
